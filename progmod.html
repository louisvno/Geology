Delta model

Language: Matlab

This model I built for a course at the uni. It is a model to calculate the distance of delta progradation over time. The main goal was to study/predict the effect of sediment supply on the progradation rate. The main variable is the sediment supply. The model simplyfies the delta shape as cylinder segments with height dy and radiuses to for the cone shape of the delta. It then calculates the time it takes to fill a new volume of a progradation step. The bathymetry was also implemented in the model.

Stockchart program

Language: Matlab

One of my hobbies is to follow the stock martkets. I wanted to have historical data but being Dutch, I didn't want to pay for it. There are lot of free sources for end of day data but it is hard to get a nice database. So I found this great script from Luminouslogic.com. to pull data from Yahoo Finance and I created a program around this to make an automated database.

The idea is that when you run the code, it searches the last date recorded in the database. If this is not equal to the most recent date available on Yahoo Finance, it gets the updates of the daily stock data from Yahoo Finance until the most recent date and adds them to the database (textfile). This file i can then load into my stockchart programm et voila i have free end of day data that is easy to update just by running the program.
